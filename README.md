# Tenable CyberAgents Exchange — Founders Pre-Launch Repository

This is a **private** repository where founding contributors submit their agent and playbook listings before the exchange . tenable . com website goes public.

Tenable employees can see the website built off of this repo in staging at https://exchange-staging.tenable.com/ — log in with your Okta credentials.

Once launched, these listings will be published on the public exchange. Until then, this repo is the staging ground for early contributions.

## What This Repo Contains

| Directory | Contents |
|-----------|----------|
| `agents/` | Markdown listing files for cybersecurity AI agents |
| `skills/` | Markdown listing files for AI coding assistant skills |
| `mcp-servers/` | Markdown listing files for MCP servers |
| `playbooks/` | Markdown listing files for multi-agent playbook workflows |
| `templates/` | Templates you'll use to create your listing file |
| `data/` | Controlled vocabularies (valid integrations, types, tiers) |
| `scripts/` | CI validation scripts |

## How to Contribute

**Read [CONTRIBUTING.md](CONTRIBUTING.md) for complete step-by-step instructions.**

The short version:

1. Create a free/personal GitHub account (if you don't have one)
2. Push your agent code to a repo in your personal account
3. Add a completed template (agent, MCP server, or playbook) to your repo
4. Get added to this repository (contact Justin Buchanan, Patrick Ramseier, or DJ Zito)
5. Clone this repo, create a branch, add your listing `.md` file, and open a pull request
